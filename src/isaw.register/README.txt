Introduction
============
NYU ISAW Registration system, this creates a sitewide user for the main web property for ISAW located at http://isaw.nyu.edu. As well as for the ticket support system and copiers. 

There are some things to note, the primary concern is that as of June 20, 2013 this will need an iterative approach. Due to the fluctuation with our current setup and because it has yet to be use-cased tested it will need adjusting as exceptions and failures are found.

If you have any questions please email isaw.it-group@nyu.edu.
